# List and Tuple

1. List
   1. Digunakan untuk menyimpan kumpulan item
   2. Mutable (bisa diubah)
   3. Ditulis dengan tanda `[]`
2. Tuple
   1. Digunakan untuk menyimpan kumpulan item
   2. Immutable (tidak bisa diubah)
   3. Ditulis dengan tanda `()`

Contoh:
```json
buah_list = ['apel', 'mango', 'banana']
angka_tuple = (1,2,3)
```
